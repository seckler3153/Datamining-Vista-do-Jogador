INSTRUÇÕES:

• Clique no botão de componente e em seguida no quadrante que quiser para inserir a imagem do botão.
• Segure "Shift" para colocar múltiplos do mesmo componente e aperte "Barra de Espaço" para cancelar a seleção de um botão.
• Ao clicar em um quadrante e arrastar para outro, é criado um caminho entre eles.
• Segure "Shift" criando um caminho para criá-lo de forma externa ao mapa.
• Os botões de Meeple e de Firewall entram no meio do mapa e podem ser arrastados pelo mapa.
• Clicar com o botão direito sobre um quadrante o faz ficar invisível, para representar quadrantes sem pastas.
• Para apagar uma imagem de um botão em um quadrante, clique duas vezes com o botão esquerdo do mouse sobre ela.
• Para apagar um caminho, clique com o botão direito do mouse ou crie outro caminho por cima, de mesmo tamanho e mesma direção.
• Para apagar um Meeple ou uma Firewall, clique com o botão direito sobre eles.
• Para qualquer problema, Ctrl + Z é seu amigo.
